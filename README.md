# Editorial Portfolio

A minimalist dark personal portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Deploy

Deploy this folder as the main Vercel project for `euel.dev`.

Set this environment variable in that Vercel project:

```bash
NEXT_PUBLIC_SITE_URL=https://euel.dev
```
